# 开发资源

我们为 iOS 和 Android 平台准备了完善的开发资源，有丰富的开发套件，涵盖硬件连接、数据解析、情感云调用和图表 UI 展示。SDK 都是独立封装，你可以根据自己 app 开发的实际需求，按需取用。

我们的 SDK 做了完善的封装，借助 SDK 可以快速上手开发。SDK 还提供了很多个性化定制的功能，详细的定制 API 可以查看各个 SDK 的 API 文档。

## SDK 汇总列表
